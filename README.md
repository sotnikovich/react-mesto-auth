# Mesto

## Описание

"Место" интерактивная страница куда пользователи могут добавлять различные фото и картинки. Проект создан в рамках учебного курса Яндекс Практикум.

## Технологии

- Flex
- Grid
- БЭМ Методология
- Webpack
- React JS
- API
- Проект подключен к серверу
